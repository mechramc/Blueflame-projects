# Animation Assets

This folder contains 3D animation assets used in the application. Ensure that all `.glb` files are placed here.

## Current Assets
- `character_animations.glb`: Stylized 3D character animations.
- `environment_animations.glb`: Stylized 3D environment animations.

## Notes
- Use optimized `.glb` files to ensure smooth performance on mobile devices.
- Test animations thoroughly to ensure compatibility with the app.
